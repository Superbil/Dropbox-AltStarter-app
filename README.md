# Dropbox-AltStarter-app
Start another Dropbox for Mac OS X

## How to use

1. Install last [release version](https://github.com/Superbil/Dropbox-AltStarter-app/releases)
2. Unzip it
3. Run `DropboxAltStarter.app`
